# dmenu-geir 5.0

Patches included:

* lineheight

This version of dmenu uses the following font:

* hack

Install Hack font:

	$ doas pkg_add hack-fonts

## Installation

Clone this repo and compile from source:

	$ git clone https://github.com/geirda/dmenu-geir.git
	$ cd dmenu-geir
	$ git checkout openbsd
	$ doas make clean install
